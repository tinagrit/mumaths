# MUMATHS _**BETA**_
is a minimal web app/game allowing users to practice mental math calculation. This project is a work-in-progress as it is being converted into React.
___
MUMATHS is heavily inspired by [Monkeytype](https://github.com/monkeytypegame/monkeytype). Thanks to [Miodec](https://github.com/miodec) and all of its contributors!
___

### Technologies
- The app runs on [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) bundled with [Vite](https://vitejs.dev/).
- Client side routing is handled by `react-router-dom`.
- Icons are from [FontAwesome](https://fontawesome.com/).

### Development
1. Install dependencies `npm install`.
2. Start a dev server `npm run dev`.
3. Build for production `npm run build` (emits to `dist/`).
4. Preview a production build locally `npm run preview`.

### Gamemodes
1. **Calculation**
    - **Operations + - x /** Two random numbers given per question. Negative answer off/on on subtraction mode.
    - **Shuffle** Operation is randomly selected.
2. **24 Game** 
    - Four numbers given per question. Apply operations in any order to make 24.  
3. **Factor**
    - Quadratic expression is given. Calculate the two roots.
